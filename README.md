# Js13kGames 2024

An entry to the [js13kGames 2024 competition](https://2024.js13kgames.com/).

## Running and building

First, install dependencies with

    $ npm install

To run the game in the browser for development:

    $ npm run dev

Building the zip package:

    $ npm run build
